# 🗺 Travel Journal

Веб-застосунок для планування подорожей та ведення цифрового щоденника мандрівника.

## Стек технологій

**Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT, Multer  
**Frontend:** React.js, React Router, SCSS, Vite, Leaflet  

## Структура проєкту
travel-journal/
├── backend/        # Серверна частина (Node.js + Express)
└── frontend/         # Клієнтська частина (React + Vite)

## Функціональність

- Реєстрація та авторизація (JWT)
- Створення та управління поїздками
- Додавання місць маршруту з геолокацією на карті
- Ведення щоденника з фото та тегами
- Публічні профілі та поїздки
- Пошук мандрівників за іменем, країною, місцем, тегом
